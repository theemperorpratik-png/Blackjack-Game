# 🃏 Blackjack Game
It is a simple and interactive version of a blackjack game made using html, css and js.
In this game(if you dont know) player tries to get close to or to 21 without going over and if a person gets 21 it is blackjack. Theres also the dealer who is the one making you bet money and he also draws two cards but one is hidden and you can decide to hit and if your card is more than dealers you win and if it is less you lose.
It is not a full fleged game because i am a beginner and js is kinda scary(took me this long to even build this).
The features it lacks: dual character of ace, cards are represented as numbers(but probablity is accurate), no dobule down or insurance feature
but it is still a fun game to test your luck and i used no ai(you can tell by the messy code)
this game was inspired from an exercise of webdev of freecodecamp but it is more advanced version as the exercise was very simple.
Its hard to get this game at first and if you and confused you can watch a tutorial in youtube.
