# 🃏 Blackjack Game

This is a simple, browser-based Blackjack game that is built using HTML, CSS, and JavaScript. I made this project to be better with the basics of JavaScript by designing the famous casino card game.

# Theme
This project adopts an indie casino arcade theme. The project attempts to replicate the feeling of a very basic digital Blackjack game, and the design is made to keep players focused on the game, with no unnecessary design or elaborate game features.

This game is made to be replayed. The design of this project focuses on the player vs dealer interaction and repeats as the player vs player interaction becomes secondary. This design is meant to prioritize the ease of understanding the game mechanics of Blackjack as well as the game’s core design features.

The focus of this Blackjack game is the simplicity of the design. The game is made in an easy, simple casino style and the design is made to be fun while still a beginner JavaScript game.

# Features

Start a new Blackjack game

Press "New card" to draw additional cards

Press "Hit" to compare with dealer's card

Blackjack and bust detection

Win/lose displayed on game board

Simple/responsive game board

Play as many rounds as you like

# How to Play
If you are new to blackjack, you and the dealer are each dealt two cards. Your aim is to try and total your hand to 21, or as close as you can without exceeding. To get a third card you press the “new card” button. If your total exceeds 21 you “bust” and lose the hand.You can also press "hit" anythime and if the dealer’s hand is lower than your total and you did not bust, you win and if your total is less than the dealers you loose. If you are dealt a total of 21 with your first two cards this is a “blackjack” and you automatically win.

# Limitations

Cards are only displayed as numbers

No dobule down or insurance feature

Lacks the dual ace functionality(1 or 11)
