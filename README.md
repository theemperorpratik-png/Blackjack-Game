# 🃏 Blackjack Game
It is a simple and interactive version of blackjack game made using html,css and js.
<hr>
Theme : It is an indie game solely created by me without anyones support with my own design. It is technically not a endless game but you can keep playing it over and over again.
<hr>
Functionality :
<hr>
Start a new Blackjack game
<hr>
Draw additional cards using the Hit button
<hr>
Automatic score calculation
<hr>
Detects:
<hr>
Blackjack (21)
<hr>
Bust (score over 21)
<hr>
Winning and losing conditions
<hr>
Simple and responsive user interface
<hr>
In this game(if you dont know) player tries to get close to or to 21 without going over and if a person gets 21 it is blackjack. Theres also the dealer who is the one making you bet money and he also draws two cards but one is hidden and you can decide to hit and if your card is more than dealers you win and if it is less you lose.

But it is still a fun game to test your luck and i used no ai(you can tell by the messy code)
